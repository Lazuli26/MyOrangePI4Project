## Storage Reality

- The system root is on `SD card`, not eMMC
- Root device: `mmcblk1p1`
- Root filesystem: `ext4`
- eMMC migration has not happened yet
- An eMMC was ordered recently but is not installed in the system yet

Why this matters:
- This board is fine for light use in its current state
- Write-heavy services should be treated carefully until a better storage target exists
